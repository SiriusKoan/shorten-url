## setup
1. Create a folder named `s` in the parent directory of app.py.
```sh
$ sudo mkdir ../s
```
2. Edit the port and IP address in app.py.
## deploy
Run app.py. You can modify the port and IP address.
```sh
$ sudo flask run --port 8080 --host 127.0.0.1
```
