## setup
1. Create a folder named `s` in the parent directory of app.py.
```sh
$ sudo touch .num.txt && echo "0" > .num.txt
```
2. Edit the port and IP address in app.py.
## deploy
Run app.py.
```sh
$ sudo python3 app.py
```
